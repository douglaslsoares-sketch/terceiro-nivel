
# Terceiro Nível

O Terceiro Nível é uma iniciativa de preparação do nível de poder
que trabalha, consome e sustenta a sociedade,
mas historicamente não se organiza como poder.

Este repositório documenta, de forma pública e versionada,
a construção contínua desse processo.

Se não nos prepararmos como poder,
seremos sempre usados como recurso.
