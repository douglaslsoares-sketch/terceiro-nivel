Três níveis, três ritmos

Existem níveis diferentes de poder funcionando ao mesmo tempo.

Os níveis de cima vivem em outro ritmo.

Eles têm tempo.

Planejam. Se organizam. Pensam no longo prazo.

O Nível 3 vive no aperto.

Corre para dar conta da sobrevivência.

Trabalha para manter a vida em pé.

Resolve o que aparece.

Nesse ritmo, não sobra tempo para se preparar.

E quando não há tempo para se preparar, a gente passa a reagir.

É aí que o erro começa.

Onde a gente erra

O problema do Nível 3 nunca foi falta de força.

Falta de força não combina com quem trabalha o dia inteiro.

O erro foi outro.

Foi acreditar que não é poder.

Enquanto a gente produz, consome e sustenta tudo,

outros se organizam para decidir os rumos.

Não porque sejam melhores.

Mas porque tratam o poder como algo que se prepara.

Preparar não é brigar

Preparar não é gritar.

Não é reclamar sem entender.

Não é bater de frente no escuro.

Preparar é parar um pouco.

É olhar a engrenagem.

É entender onde a nossa força entra e para onde ela vai.

Quando a gente entende,

a mesma força rende mais.

Instrumentos, não discurso

O Terceiro Nível não nasceu para fazer discurso.

Aqui não tem promessa.

O que existe são instrumentos.

Textos. Conversas. Projetos.

Tudo para ajudar a enxergar o que normalmente passa batido.

Se não nos prepararmos como poder, seremos sempre usados como recurso.

Projeto em construção

O Terceiro Nível está sendo construído aos poucos.

Tudo é público.

Tudo fica registrado.

Acompanhar faz parte.

Porque entender junto já é uma forma de se preparar.
