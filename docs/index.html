TERCEIRO NÍVEL

Quem faz o país funcionar precisa entender como o país funciona.

SOBRE

O Terceiro Nível é a base da sociedade.

É onde estão milhões de pessoas que trabalham, consomem e pagam impostos.

Essa base garante o funcionamento do país.

Mas raramente observa como a estrutura que organiza a sociedade funciona.

Este espaço existe para observar essa estrutura.
