Projeto editorial de observação

TERCEIRO NÍVEL

Quem faz o país funcionar
precisa entender
como o país funciona.

É um projeto para ajudar quem sustenta o país a entender como o país funciona.

Você está na base

O que é este projeto

O Terceiro Nível parte de uma ideia simples.

Existe uma parte da sociedade que faz tudo funcionar.

Essa parte trabalha, consome e paga impostos.

Essa parte sustenta o país.

Mas quase nunca vê a estrutura

Ao mesmo tempo em que sustenta tudo,
essa base quase nunca consegue observar
como o país funciona de fato.

As decisões aparecem.
As notícias aparecem.
Os conflitos aparecem.

Mas o funcionamento por trás disso tudo
raramente fica claro.

É aí que este espaço entra

O Terceiro Nível não existe
para fazer barulho.

Existe para observar.

Observar como o país se organiza.
Como o dinheiro circula.
Como as decisões são tomadas.
E quem influencia essas decisões.

Um outro ponto de vista

Este projeto é um ponto de observação.

Um lugar para olhar o que acontece no país
não só pelo que aparece na superfície,
mas pela estrutura que faz aquilo acontecer.

A questão central

Quando você começa a observar isso,
percebe uma coisa importante:

quem sustenta o sistema
nem sempre entende o sistema.

E, muitas vezes,
acaba reagindo sem conseguir enxergar
o que está por trás do que aparece.

Por que isso importa

Entender a estrutura
não é um detalhe.

Isso muda a forma de ver.
Muda a forma de interpretar.
Muda a forma de reagir.

E pode mudar até a forma
de escolher o que serve
e o que não serve para a base.

Como o site funciona

Aqui não é notícia.

Aqui é leitura.

Leitura do que acontece no país
a partir da estrutura
e de como ela afeta a vida
de quem está na base.

Por isso o site é feito em blocos curtos,
como pontos de observação.

E as camisetas?

As camisetas fazem parte disso.

Não são só produtos.

São extensões do projeto.

Levam a mensagem para a rua,
abrem conversa,
provocam pergunta
e convidam mais gente
a olhar para a estrutura.

No fundo, é sobre isso

Quem sustenta o país
precisa entender
como o país funciona.

O Terceiro Nível existe para isso:

ajudar quem está na base
a enxergar a estrutura
que organiza a vida no país.

Você está na base.

E, quando a base entende,
alguma coisa começa a mudar.

Com você muda.
