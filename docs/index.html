TERCEIRO NÍVEL

CARREGA O PIANO
MAS NÃO TOCA A MÚSICA

Quem faz o país funcionar precisa entender como o país funciona.

Você está na base

Toda a sociedade trabalha,
consome
e paga impostos.

O país funciona a partir disso.

Mas o peso não é igual.

Na base, o peso é maior.

A base garante o funcionamento
no dia a dia.

E, muito provavelmente,
você está nela.

Quem está na base
sente mais o que acontece no país.

Mas quase nunca consegue ver
como o país funciona.

Você vê decisões.
Você vê notícias.
Você vê conflitos.

Mas o que aparece
não mostra o todo.

Quando você não entende
como o país funciona,
você reage ao que aparece.

Sem enxergar
o que está por trás,
reage sem ver o todo, no escuro.

Sem a base,
o país não funciona.

Mas quem garante o funcionamento
quase nunca participa
com o mesmo peso das decisões.

Carrega o piano,
mas não toca a música.

Este projeto existe para marcar posição.

Mostrar o país
a partir de quem sustenta o funcionamento.

Dar clareza
para quem carrega mais peso.

Quem carrega mais
precisa entender mais.

Porque sem entender,
a base sustenta sem direção.

E quando entende,
começa a escolher melhor.

Quando você entende
como o país funciona,
alguma coisa muda.

Aqui não é notícia.

Aqui é leitura.

Não é só o que aconteceu.

É por que aconteceu.

Você está na base.

E quando a base entende,
começa a escolher.

Com você muda.
