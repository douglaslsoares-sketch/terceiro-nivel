Como isso começou
Tudo começou depois de uma palestra sobre a dívida pública.
O que eu entendi ali me incomodou.
Parecia que uma parte enorme do dinheiro público ia para o sistema financeiro — para pagar juros da dívida.
enquanto faltava para saúde e educação.
O governo pega dinheiro, emite títulos, e depois devolve com juros.
Até aqui, nada de errado.
O problema começa quando vira um ciclo:
dívida nova para pagar dívida velha.
Cada vez mais dinheiro vai para isso.
E falta para:
saúde
educação
infraestrutura
É aí que o país trava.
Existe uma proposta de auditoria dessa dívida.
Mas nunca foi feita de verdade.
