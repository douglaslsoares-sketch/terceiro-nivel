Manifesto

Este manifesto continua a conversa iniciada na página principal. Aqui não tem explicação longa. É só a posição que orienta tudo o que vem depois.

Depois de olhar para a própria vida, fica mais fácil perceber uma coisa.

Existe um nível que sustenta tudo, mas quase nunca se vê como poder.

É o nível que trabalha.

É o nível que consome.

É o nível que produz.

Sem ele, nada funciona.

Mesmo assim, vive correndo, reagindo ao que aparece, tentando sobreviver.

Enquanto isso, outros níveis se organizam.

Planejam.

Antecipam.

Definem o ritmo.

Não porque sejam melhores.

Mas porque tratam o poder como algo que se prepara.

O maior erro do Nível 3 nunca foi falta de força.

Foi acreditar que não é poder.

Quando a gente não se reconhece como poder, passa a funcionar só como recurso.

Recurso que trabalha.

Recurso que consome.

Recurso que sustenta decisões que não ajudou a tomar.

Este manifesto não pede nada.

Não promete nada.

Não convoca ninguém.

Ele apenas afirma uma posição.

O Terceiro Nível existe para que quem sustenta também compreenda.

Preparar não é brigar.

Não é gritar.

Não é confrontar no escuro.

Preparar é entender a engrenagem.

É reconhecer a própria força.

É parar de abaixar a cabeça sem saber por quê.

O que falta ao Nível 3 não é vigor.

É consciência organizada.

Este não é um projeto para salvar ninguém.

É um processo para reconhecer o lugar que a gente já ocupa.

Se não nos prepararmos como poder, seremos sempre usados como recurso.
