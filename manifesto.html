Manifesto

Este manifesto faz parte da conversa iniciada na página principal do Terceiro Nível. Ele expressa, de forma direta, a posição que orienta todo o restante do trabalho.

Depois da constatação apresentada na página inicial, fica mais fácil perceber que existe um nível de poder que sustenta toda a sociedade, mas raramente se reconhece como poder.

É o nível que trabalha. É o nível que consome. É o nível que produz.

Sem ele, nada funciona. Ainda assim, vive reagindo, correndo atrás, tentando sobreviver.

Enquanto isso, outros níveis se organizam. Planejam. Antecipam. Se preparam.

Não porque sejam melhores. Nem mais numerosos. Mas porque tratam o poder como algo que se constrói com método.

O erro histórico do terceiro nível sempre foi acreditar que não é poder.

Quando não nos reconhecemos como poder, passamos a funcionar apenas como recurso.

Recurso que trabalha. Recurso que consome. Recurso que sustenta privilégios que não escolheu.

Este manifesto não é um pedido. Não é uma promessa. Não é uma convocação emocional.

É uma afirmação de posição.

O Terceiro Nível existe para preparar o nível de poder que sempre sustentou o sistema, mas nunca foi convidado a compreendê-lo.

Preparação não é confronto. Não é grito. Não é negação da realidade.

Preparação é compreensão. É leitura da engrenagem. É reconhecimento das próprias forças.

Nossas forças são orgânicas: pessoas, trabalho, consumo, comunicação cotidiana, circulação de ideias.

O que falta não é força. É consciência organizada.

Este não é um projeto para salvar ninguém. É um processo para que cada um reconheça o lugar que já ocupa.

se não nos prepararmos como poder, seremos sempre usados como recurso.
